# ジャッジ君 (old project)
作成：2017年

「〇〇しようかな、どうしようかな」というときに「やっちゃえ」「やめとけ」と判定してくれます。  
（例：今22:00だけどアイス食べたいかも）

### 操作
E：判定する  
R：リセットする

### 仕組み
6面サイコロを9つ同時に1回だけ振ります。  
出目の合計値の1の位もしくは10の位が、当日の月もしくは日の1桁目もしくはその倍数と一致するとき、やっちゃえ判定が出ます。
